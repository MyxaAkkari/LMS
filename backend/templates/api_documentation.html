API Documentation

POST /signup

Register a new user. Requires: username, email, password, city, age, and account type (admin/user).

Returns user details on success.
POST /login

Authenticate a user. Requires: email and password.

Returns an access token on success.
GET /get-user-info

Retrieves details of the currently authenticated user.

Returns user details including username, email, city, age, and account type.
GET /books

Retrieve a list of all books.

Returns a list of books.
POST /books/add

Adds a new book to the library (Admin only). Requires: name, author, year_published, description, loan_type, image (optional).

Returns confirmation of the book addition.
PUT /books/edit/{book_id}

Edits details of an existing book (Admin only). Requires: name, author, year_published, description, loan_type, image (optional).

Returns confirmation of the book update.
DELETE /books/delete/{book_id}

Deletes a specific book from the library (Admin only).

Returns confirmation of the book deletion.
POST /loan/{book_id}

Loans a book to the current user. Requires: book_id.

Updates book status and returns confirmation.
POST /return/{loan_id}

Returns a loaned book. Requires: loan_id.

Updates book status and returns confirmation.
GET /user/loans

Retrieves all books loaned by the current user.

Returns a list of loaned books.
GET /admin/loans

Fetches all loaned books across all users (Admin only).

Returns a list of all loans including book details and user IDs.
GET /customers

Retrieves a list of all customers/users (Admin only).

Returns a list of users with their details.
GET /customers/{user_id}

Fetches details of a specific user by their ID (Admin only).

Returns details of the specified user.
DELETE /customers/{user_id}

Deletes a specific user by their ID (Admin only).

Confirms deletion of the user.
